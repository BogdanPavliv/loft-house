## Верстка сайту Loft House

- git,
- gulp,
- npm,
- figma, optimized graphics, svg where it is possible
- html,
- scss,
- gulp,
- js
- flex boxes
- grid
- BEM naming

Documentation and usage examples inside.

Install dependencies:
```
npm i
```

Run in development mode:
```
gulp
```

To create production version use:
```
gulp building
```
